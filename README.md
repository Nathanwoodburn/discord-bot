# discord-bot

### About the Bot
The Woodburn Bot responds to some simple commands and to greatings.


### Commands

| Commands  | Response                |
|-----------|-------------------------|
| Hello     | Random Greating         |
| Hi        | Random Greating         |
| !roll     | Rolls A Die             |
| !flip     | Flips A Coin            |
| !ping     | Gets the bot latency    |
| !commands | Gets a list of commands |

Send Suggestions to bot@woodburn.id.au
